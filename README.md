# 🔐 XCrypt Encryption

> **Secure. Private. Encrypted.**
>
> XCrypt Encryption is a modern privacy-first platform that allows users to securely encrypt, store, and share files with end-to-end encryption, optional password protection, expiring links, and access codes.

---

## 🌐 Live Demo

**Website:** https://xcrypt-encryption.bolt.host/

---

# 📖 Overview

XCrypt Encryption is designed to make secure file sharing simple, fast, and reliable.

Instead of uploading files directly to the cloud in plain form, XCrypt encrypts them before they are shared, ensuring that only authorized users can access the original content.

The project focuses on:

* Privacy
* Security
* Ease of use
* Modern user experience
* Zero unnecessary exposure of files

Whether you're sharing confidential documents, personal photos, or sensitive files, XCrypt provides a safer alternative to traditional file-sharing services.

Protect files with an optional password.

---

## ⏳ Expiring Links

Automatically expire files after:

* Custom duration
* Number of downloads
* Manual deletion (future)

---

## 🎟 Unique Access Codes

Every uploaded file receives a unique access code.

Example:

A25Y-MW67-9XNU

This makes sharing easier while improving security.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/xcrypt-encryption.git
```

Install dependencies:

```bash
npm install
```

Run locally:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
